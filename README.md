# Monitor de Investimentos Privados de Contagem

Sistema para monitoramento automático de:

- Investimentos privados
- Expansões empresariais
- Novas empresas
- Licenças ambientais
- Decretos
- Leis
- Incentivos fiscais
- Obras e empreendimentos

## Fontes monitoradas

- Prefeitura de Contagem
- Diário Oficial de Contagem
- Agência Minas
- Invest Minas
- FIEMG

## Estrutura

- keywords.txt → palavras-chave
- empresas.txt → empresas monitoradas
- sites.txt → fontes monitoradas
- dados/ → base histórica
- importador_excel.py → leitura da planilha histórica

## Objetivo

Construir uma base histórica e um monitor automático de investimentos privados no município de Contagem.
